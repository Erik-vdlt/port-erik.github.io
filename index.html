Proyectos

Veterinaria

Proyecto de software para escritorio sobre una veterinaria para registro de mascotas, dueños y medicamentos que son necesiarios. Las funciones del software debian ser un registro de informacion, eliminarla, hacer busquedas y actualizar la informacion.

Tecnologias Usadas

  1. Pyqt4
  2. Python
  3. Mysql
Repositorio

Hotel

Proyecto CRUD para un hotel donde se registraban los huespedes y hacen reservaciones para su estancia, el proyecto fue hecho en netbeans, donde se hizo la interfaz grafica y registrar los datos en base de datos en PostgreSQL.

Tecnologias Usadas

  1. Java
  2. Netbeans
  3. PostgreSQL

Teatro

Un proyecto web sobre un teatro que registra las obras que estaran disponibles, un CRUD y un login para registro de usuarios y un captcha. Dentro podemos registrar las obras de teatro que se presentaran para verano y que actores interpretaran los papeles.

Tecnologias Usadas

  1. JavaScript
  2. HTML5
  3. CSS
  4. Bootstrap
  5. PHP

Cactus

Proyecto web personal de un crud para el registro de series animadas en una base de datos sqlite3, en la parte de backend se utiliza python flask y alchemy, en la parte de front end se utiliza bootstrap.

Tecnologias Usadas

  1. sqlite3
  2. flask
  3. alchemy
  4. bootstrap
